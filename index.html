Control de Ángulo con Motor y Raspberry Pi Pico

Un proyecto para controlar la aguja mediante engranajes y un motor paso a paso

Descripción del Proyecto

Este proyecto consiste en un control de ángulo de una aguja mediante un motor paso a paso, utilizando una Raspberry Pi Pico para el control. El sistema se basa en un conjunto de engranajes, uno grande conectado al motor y otro pequeño que mueve la aguja. El control es altamente preciso, permitiendo ajustar el ángulo de la aguja con gran exactitud. La Raspberry Pi Pico se encarga de interpretar las señales y gestionar el movimiento del motor.

Componentes

  • Raspberry Pi Pico
  • Motor paso a paso
  • Engranaje grande
  • Engranaje pequeño
